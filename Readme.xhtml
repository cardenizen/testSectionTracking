In addition to plug-in dependencies (see application.properties) this code depends on:

Java         JRE/JDK 1.6.x
Groovy    JDK 1.7.x
Grails       1.3.7

Running

Run the app in the development environment using development DB connection
grails run-app -XX:PermSize=64m -XX:MaxPermSize=128m

Run the app in the development environment using test jndi DB connection
grails test run-app -XX:PermSize=64m -XX:MaxPermSize=128m -Dpassword=DEV_PASSWORD

Run Integration Tests
grails test test-app -XX:PermSize=64m -XX:MaxPermSize=128m -Dpassword=DEV_PASSWORD
